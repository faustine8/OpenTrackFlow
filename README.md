# OpenTrackFlow

An event-driven container tracking platform built around DCSA-compatible shipment events.

## Overview

OpenTrackFlow is a DCSA-first container tracking and traceability platform that ingests standardized
shipment events,
validates data quality, builds container timelines, and publishes normalized tracking updates to
downstream systems.

## Core Features

- Event ingestion
- DCSA-compatible normalization
- Data quality validation
- Container timeline aggregation
- Subscription and webhook delivery
- Event traceability
